   G4.9 FCT XML 测试日志看板

这是一个用于看板进行展示。这是一个用于解析、监控和排查 G4.9 FCT 测试 XML 日志的轻量级 Web 工具。

项目的核心目标是：  
让工程师不用手动打开超长 XML 文件，也可以快速查看最近测试记录、查询指定 SN、定位 FAIL 项、识别中断测试，并统计高频 FAIL 项。

---
   
   1. 项目简介

在 FCT 测试过程中，每台被测产品通常都会生成一个 XML 测试结果文件。真实的 FCT XML 文件通常包含大量测试节点，单个文件可能有几百个测试项，人工打开和查找问题非常低效。

本工具通过自动解析 XML 文件，把关键信息整理成网页看板，包括：

- 总测试记录数量
- PASS 数量
- FAIL 数量
- 暂停数量
- 中断数量
- 解析异常数量
- 最近测试记录
- TOP FAIL 项
- 指定 SN 查询
- 弹窗查看测试详情
- 原始测试项明细滚动查看
- 根据 SN 前缀识别产品型号
